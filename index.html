Ski Cam Analytics – AI analýza kamerových dat

MVP aplikace pro počítání lidí v lyžařském areálu z HLS video streamu. Detekuje a trackuje osoby pomocí YOLO ONNX modelu a zobrazuje metriky v reálném čase na webovém dashboardu.

  • Detekce osob přes YOLO ONNX + SORT-like tracker s Kalman filterem
  • Occupancy counter a počítání průchodů čárou (line crossing)
  • FastAPI backend s WebSocket broadcastem + Chart.js dashboard

Geo Places Quiz – interaktivní geografický kvíz

Webová GIS aplikace pro procvičování znalostí obcí s rozšířenou působností ČR (206 ORP) formou interaktivní mapy. Hodnotí přesnost kliknutí s koeficientem 0,5–1,0 dle vzdálenosti od centroidu.

  • Leaflet mapa s PostGIS backendem – data ČÚZK, S-JTSK → WGS84
  • Filtrování podle kraje a okresu, statistiky v localStorage
  • Node.js/Express API s kešováním a GZIP kompresí
JavaScript Leaflet Node.js PostGIS Express
